# qbot game statistics plugin
This plugin gets game statistics from game of your choice and will post it in an embed, this includes Player Count, Visited Count, Favorited Count, Upvote Count, and Downvote Count.

# Instructions
* Add the MIT license into your editor.
* Create a file named gamestats.js inside of your commands folder.
* Copy and paste code from gamestats.js into your editor.
* Replace 'gameId' with the ID of your game.
